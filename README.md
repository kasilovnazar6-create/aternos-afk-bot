## 📖 О проекте

**Aternos AFK Bot** — это специализированный Minecraft бот, созданный для решения главной проблемы бесплатных серверов Aternos: автоматического выключения при отсутствии игроков онлайн.

### 🤔 Почему Aternos выключает серверы?

Aternos предоставляет бесплатный хостинг Minecraft серверов, но с одним важным ограничением:

- **Сервер автоматически выключается через 5-10 минут после того, как последний игрок покидает его**
- **Сервер перезагружается каждые 24 часа**
- **Есть защита от бездействия (AFK kick)**

Этот бот решает все эти проблемы, имитируя реального игрока с естественным поведением.

### 🎯 Как бот решает проблему?

Наш бот использует библиотеку **Mineflayer** для создания виртуального игрока, который:

1. **Подключается к серверу** автоматически после его запуска
2. **Имитирует реального игрока** через систему движений
3. **Обходит AFK-защиту** сервера
4. **Автоматически переподключается** при обрыве связи
5. **Перезаходит каждые 2 часа** для обхода ограничений Aternos

## 📦 Установка

### 📋 Предварительные требования

Перед установкой убедитесь, что у вас установлено:

1. **Node.js** версии 16 или выше
   - [Скачать с официального сайта](https://nodejs.org/)
   - Проверить версию: `node --version`

2. **Git** (опционально, для клонирования)
   - [Скачать Git](https://git-scm.com/)

3. **Aternos сервер**
   - [Создать сервер](https://aternos.org/)

### 🔧 Локальная установка

#### Способ 1: Клонирование репозитория

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ВАШ_ЛОГИН/aternos-afk-bot.git

# 2. Перейдите в папку проекта
cd aternos-afk-bot

# 3. Установите зависимости
npm install

# 4. Настройте settings.json (см. раздел Настройка)

# 5. Запустите бота
npm start

⚙️ Настройка
📄 Файл settings.json
Все настройки бота находятся в файле settings.json. Вот подробное описание каждого параметра:

json
{
  "server": {
    "ip": "creativno.aternos.me",     // IP вашего Aternos сервера
    "port": 55758,                     // Порт сервера (указан в панели Aternos)
    "version": "1.20.1"               // Версия Minecraft (важно указать точно!)
  },
  "bot": {
    "username": "INDERFIF",           // Имя бота на сервере
    "auth": "offline",                // Тип авторизации (offline/microsoft)
    "autoAuthPassword": "chalol78"    // Пароль для /login (если есть плагин авторизации)
  },
  "antiAFK": {
    "enabled": true,                  // Включить анти-AFK систему
    "sneakInterval": 30000,           // Интервал приседания (миллисекунды)
    "jumpInterval": 15000,            // Интервал прыжков (миллисекунды)
    "lookInterval": 5000,             // Интервал вращения камеры (миллисекунды)
    "walkEnabled": true,              // Включить ходьбу
    "walkSpeed": 3000                 // Интервал смены направления ходьбы (миллисекунды)
  },
  "reconnect": {
    "enabled": true,                  // Включить авто-переподключение
    "initialDelay": 10000,            // Начальная задержка перед реконнектом (10 сек)
    "maxDelay": 60000                 // Максимальная задержка (60 сек)
  }
}
